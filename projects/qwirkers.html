Qwirkers

Easy game to pick up, hard to put down

Qwirkers is a mobile game based on the famous tile game by MindWare, Qwirkle. The game is easy to pick up and addictive. The more you play, the more your tactical maneuvers will help you win.

Discovery & Strategy

Time to flip the tables

The development process began with the focus on designing a modern and intuitive UI. Extensive research and multiple iterations led to a clean and customizable UI that is easy to navigate. Incorporating design elements such as simple color schemes, minimalistic icons, and a logical layout made the UI aesthetically pleasing.

The main functions were moved to the lower area to make it easy to use with one hand.

Not just another mobile game

As we mentioned, gameplay wasn't everything. Therefore, we decided to provide users with the ability to choose custom avatars or create one from scratch, allowing them to express themselves online.

Players can create a peep that is unique to them.

The game reflects a product that combines simplicity, uniqueness, and visual enjoyment in interaction.

Playing to win the game

Within the first month, the games had an average of 128 active daily users. We were able to achieve this by constantly gathering feedback from users and communicating with them to ensure that their needs were being met. As a result, we were able to achieve a daily retention rate of 37.4%.
